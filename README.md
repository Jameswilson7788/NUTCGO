## Clone
```bash
git clone https://github.com/explooosion/NUTCGO.git
```

## Into Project
```bash
cd NUTCGO
```

## Install
```bash
npm install
```

## task
```bash
npm install -g gulp
```


## Deploy
```
gulp ex-all
```

## Run
```bash
npm start
```

Restore from NUTCGO_db.bak to MSSQL
